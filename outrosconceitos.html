Internet

 A internet é a rede de computadores que troca dados e mensagens utilizando um tipo de protocolo como o modelo TCP/IP ou OSI.

Outros conceitos

Endereço MAC: (Media Access Control ou Controle de Acesso de Mídia) é um endereço físico e único, que é associado à interfaces de comunicação utilizadas em dispositivos de rede. A identificação é gravada em hardware por fabricantes de placa de rede, tornando-se posteriormente, parte de equipamentos como computadores, roteadores, smartphones, tablets, impressoras de rede e diversos outros equipamento que usam comunicação em rede.

DNS: (Domain Name System, ou sistema de nomes de domínios) é responsável por localizar e traduzir para números IP os endereços dos sites que digitamos nos navegadores.

DHCP: (Dynamic Host Confiuration Protocol) que significa Protocolo utilizado em redes de computadores que permite maquinas obterem um endereço de IP dinâmico e automático. Ele distribui endereços IP diferentes para todos os computadores que estão conectados a rede, destribuição feita em um intervalo que já foi definido e configurado no servidor.

Firewall: É um dispositivo de redes de computadores que é responsável por garantir a segurança de um ponto de rede.

Não orientado a conexão: Sem confiabilidade, já que não há garantia de envio/recebimento de pacotes.

Ethernet: É uma arquitetura de interconexão para redes locais - Rede de Área Local (LAN) - baseada no envio de pacotes. Ela define cabeamento e sinais elétricos para a camada física.

CMD: O Prompt de Comando é um interpretador do Windows, ele executa varios comandos como por exemplo: ipconfig, que é possível visualizar seu endereço IP e outras informações na rede.

Token Ring: É um protocolo de redes que utiliza uma topologia lógica de anel e funciona na camada física. Ela possui hubs inteligentes para funcionar com 8 portas ligadas entre si.

HTML: (Hypertext Markup Language, Linguagem de Marcação de Hipertexto) é uma das linguagens que utilizamos para desenvolver websites.
