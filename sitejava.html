Java

 Java é uma linguagem de programação orientada a objetos de alto nível e muito famosa. Este é um tutorial básico do básico apenas para mostrarmos a linguagem de programação Java, usaremos a plataforma online Repl.it - clique aqui, pois não será necessário baixar programas para compilar está linguagem.

Entrando no site acima, vemos:

 A função se inicia com class main{ }, todo conteúdo deverá ficar entre as chaves, proseguindo, temos um documento público, estático e sem parâmetros, ou seja, não retornará nenhum valor e mostrará uma String, que é um texto que será mostrado para o usuário.

O comando para mostrar algo na tela é System.out.println(" texto aqui "); não esqueça do ponto e virgula no final de cada comando, é de extrema importancia.
