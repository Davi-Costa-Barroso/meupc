Hardware

 O Hardware é a parte fisica do computador, o equipamento, é onde você consegue tocar. Exemplos: Monitor, memória ram, HD, gabinete, etc...

Gabinete: É onde todos os componentes do computador ficam guardados. Pode ser considerado o corpo.


Fonte de alimentação: É um dispositivo que transforma a energia recebida da rede elétrica para as voltagens utilizadas pelos diferentes componentes do computador. Se existe um componente absolutamente vital para o funcionamento de um computador, este é a fonte de alimentação. Sem ela, o computador é apenas uma caixa de plástico e metal sem função.

Placa de rede: Sua função é controlar todo o envio e recebimento de dados através da rede.

Memória Ram(Random Access Memory): Ela armazena informações basicas que vem do HD para o processador. Considerada memória volátil, ou seja, armazena informações temporariamente.

Processador: É a unidade central de processamento(CPU), fica encaixado no soquete. É ela quem realiza as tarefas, operações matemáticas e elaboração de textos.

Bios(Sistema Básico de Entrada/Saída): Contém um pequeno software responsável por controlar o uso dos dispositivos e mantém informações de data e hora.

Bateria: Uma bateria pequena de lítio que tem a função de manter informações armazenadas enquanto o computador está desligado. Exemplo: deixa o relógio interno funcionando.

Cooler: Os processadores aquecem muito quando estão efetuando tarefas, para evitar a queima ou possíveis danos ao componente, é preciso resfriá-lo. O item-chave nessa hora é o cooler(refrigerador).

Barramento: É o encaixe de que cada peça precisa para funcionar corretamente. Há barramentos específicos para praticamente todos os componentes conectados ao sistema.

Nobreak: Serve como fonte de energia, quando falta energia o computador continua ligado para que o usuário possa salvar seus arquivos. Dependendo do Nobreak, o computador consegue ficar 30 minutos ligado.

Estabilizador: Mantém a energia do computador equilibrada(estabilizada), mesmo que aconteça uma queda de energia os componentes do computador não serão danificados.

Outros componentes:


- Placa mãe
- HD e SSD
- ChipSet
