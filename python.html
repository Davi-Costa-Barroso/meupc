Python

 É uma linguagem de programação de alto nível, interpretada e orientada a objetos. Para começar a programar você precisa instalar um programa especifico ou simplesmente programar em uma plataforma online, nesse tutorial usaremos a plataforma online Repl.it - clique aqui. Com ela não precisaremos baixar nada.

 Este é um tutorial básico para mostrar um pouco da linguagem de programação Python. O primeiro comando que mostraremos é o famoso Hello World, que significa Olá mundo. É simples, basta digitar o comando: input("ola mundo"); e apertar na seta para rodar o código ao lado.

Input significa entrada, nela, temos parêntese com aspas e dentro temos o texto que será mostrado.

Para somar, subtrair, multiplicar ou dividir dois números devemos tirar as aspas de dentro do parêntese e colocar os valores, exemplo: input(1+2);

Também podemos interagir com o usúario, vamos pedir dois numeros inteiros, somar e mostrar o resultado. digite o código abaixo:

numero1 = int(input("diga um numero: ")); numero2 = int(input("diga outro numero: ")); soma = numero1 + numero2; input(soma);

 Observa-se que declaramos três variáveis "numero1", "numero2" e "soma", utilizamos o input para pedir dois números e colocamos o comando "int" que significa que o número será inteiro, mandamos o numero que foi digitado para as suas respectivas variáveis através do sinal de igual = , depois somamos o numero 1 com o numero 2 e atribuimos o resultado para a variável "soma", por último, mostramos o resultado da variável soma.
